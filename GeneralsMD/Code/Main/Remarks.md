# Main文件简述

### app.manifest

该 XML 文件是一个 Windows 应用程序清单，其核心目的是配置应用程序在 Windows 系统上的安全权限、现代界面样式和高 DPI 显示兼容性。它要求程序以标准用户权限运行，启用 Windows Vista 风格的现代控件外观，并声明支持高级的每监视器 DPI 缩放功能（PerMonitorV2），以确保在多种不同分辨率和高 DPI 的显示器上都能获得清晰、比例正确的视觉体验。

### RTS.RC

该文件是Windows应用程序的资源定义脚本，核心作用是将图标资源和应用程序清单打包到最终的可执行文件中。它与之前分析的清单文件配合工作：清单文件声明应用程序的DPI感知和权限要求，而此资源脚本确保这些声明被实际嵌入到程序中。整体上，这确保了应用程序具有正确的图标显示，并在现代Windows系统上获得优化的高DPI显示效果和适当的权限控制。

### resource.h

该文件是Windows应用程序资源管理的核心定义文件，它将代码中需要引用的资源（如图标、位图）与具体的数字ID建立映射关系。通过与对应的.rc资源脚本文件配合，确保程序能够正确加载和使用这些图形资源。文件开源的GPLv3许可证声明表明这是一个基于原版游戏的社区开发或重构项目。
